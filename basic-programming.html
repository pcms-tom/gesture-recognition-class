任務 2：用 Python 開啟攝影機

Python 透過 OpenCV 讀取攝影機。第一次執行時，系統可能會要求允許 Python 或程式編輯器使用攝影機。

# 檔名：02_camera.py
import cv2

cap = cv2.VideoCapture(0)

while True:
    success, frame = cap.read()
    if not success:
        break

    cv2.imshow("Camera", frame)

    if cv2.waitKey(1) & 0xFF == ord("q"):
        break

cap.release()
cv2.destroyAllWindows()

學生要完成的事

  1. 把程式存成 02_camera.py。
  2. 執行 python 02_camera.py。
  3. 看到攝影機視窗後，按 q 關閉。
  4. 如果畫面沒有出現，檢查攝影機權限或把 VideoCapture(0) 改成 VideoCapture(1)。
完成標準：電腦出現即時攝影機畫面，按 q 能關閉視窗。

任務 3：加入 MediaPipe Hands

MediaPipe 負責把攝影機畫面分析成手部 landmarks；OpenCV 負責讀取和顯示影像。

# 檔名：03_mediapipe.py
import cv2
import mediapipe as mp

mp_hands = mp.solutions.hands
hands = mp_hands.Hands(
    max_num_hands=1,
    min_detection_confidence=0.7,
    min_tracking_confidence=0.7
)

print("MediaPipe Hands 已建立")

學生要完成的事

  1. 建立 03_mediapipe.py。
  2. 執行後確認終端機顯示 MediaPipe Hands 已建立。
  3. 找出程式中限制「最多偵測一隻手」的參數。
  4. 把信心門檻改成 0.5 試一次，再改回 0.7。
完成標準：程式沒有 import error，代表 MediaPipe 已正確安裝。

任務 4：偵測並畫出 21 個關鍵點

MediaPipe 需要 RGB 圖片；OpenCV 讀進來的是 BGR，所以要先轉色彩格式。

# 檔名：04_landmarks.py
import cv2
import mediapipe as mp

mp_hands = mp.solutions.hands
mp_draw = mp.solutions.drawing_utils
hands = mp_hands.Hands(max_num_hands=1)

cap = cv2.VideoCapture(0)

while True:
    success, frame = cap.read()
    if not success:
        break

    rgb = cv2.cvtColor(frame, cv2.COLOR_BGR2RGB)
    results = hands.process(rgb)

    if results.multi_hand_landmarks:
        for hand_landmarks in results.multi_hand_landmarks:
            mp_draw.draw_landmarks(
                frame,
                hand_landmarks,
                mp_hands.HAND_CONNECTIONS
            )

    cv2.imshow("Hand Landmarks", frame)
    if cv2.waitKey(1) & 0xFF == ord("q"):
        break

cap.release()
cv2.destroyAllWindows()

學生要完成的事

  1. 建立 04_landmarks.py。
  2. 執行後把手放到攝影機前。
  3. 確認手上會出現點和線。
  4. 找出哪一行負責把 BGR 轉成 RGB。
完成標準：畫面中的手會出現 MediaPipe 骨架。

常見問題與解法

遇到錯誤時，先對照錯誤訊息，再按下面方法處理。不要一次改很多地方，每次只改一件事再重新執行。

問題 1：輸入 python 後沒有反應或顯示找不到

  1. 先試：py --version
  2. 如果 py 可以用，後面的指令可改成 py 檔名.py。
  3. 如果兩個都不行，表示電腦未正確安裝 Python，重新安裝 Python，安裝時勾選 Add Python to PATH。

問題 2：pip 找不到或安裝到錯的地方

  1. 不要單獨輸入 pip install ...。
  2. 使用：python -m pip install opencv-python mediapipe
  3. 如果用的是 py，則輸入：py -m pip install opencv-python mediapipe

問題 3：安裝時顯示 pip 版本太舊

  1. 先更新 pip：python -m pip install --upgrade pip
  2. 再重新安裝：python -m pip install opencv-python mediapipe
  3. 如果學校網絡很慢，等待一段時間，不要重複快速按 Enter。

問題 4：ModuleNotFoundError: No module named 'cv2'

  1. 代表 OpenCV 沒有安裝到目前正在執行的 Python。
  2. 執行：python -m pip install opencv-python
  3. 再執行：python 01_test.py

問題 5：ModuleNotFoundError: No module named 'mediapipe'

  1. 代表 MediaPipe 沒有安裝到目前正在執行的 Python。
  2. 執行：python -m pip install mediapipe
  3. 如果安裝失敗，檢查 Python 版本，建議使用較新的 64-bit Python。

問題 6：攝影機視窗沒有出現

  1. 確認沒有其他程式正在使用攝影機。
  2. 檢查系統是否允許 Python 或程式編輯器使用攝影機。
  3. 把 cv2.VideoCapture(0) 改成 cv2.VideoCapture(1) 再試。

問題 7：攝影機視窗出現但畫面是黑的

  1. 等 3 到 5 秒，部分攝影機啟動較慢。
  2. 確認鏡頭沒有被遮住。
  3. 改用 VideoCapture(1) 或重新插拔外接攝影機。

問題 8：按 q 無法關閉視窗

  1. 先點一下攝影機畫面視窗，讓鍵盤焦點在視窗上。
  2. 再按 q。
  3. 如果仍不能關閉，到終端機按 Ctrl + C 停止程式。

問題 9：手放進畫面但沒有骨架

  1. 讓手完整出現在畫面中，不要太近。
  2. 增加光線，避免背景太亂。
  3. 把 min_detection_confidence 從 0.7 改成 0.5 試一次。

問題 10：程式縮排錯誤

  1. Python 很重視縮排，while、if、for 下一行要往右縮排。
  2. 同一層縮排要一致。
  3. 建議用 4 個空格，不要混用 Tab 和空格。
